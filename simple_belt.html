Simple Track Calculator

Introduction

This calculator aims to simplify the creation of tracks for small remote controlled vehicles. Currently it handles the simple case where there is a flexible belt wrapped around two equal size wheels.

Input Parameters

Sometimes you want to specify a system based on the belt and wheels you already have. Sometimes you want to size some wheels to make the wheels have a certain spacing. To change between these, you can adjust the "solver mode" which will change the set of input parameters.

If you have an existing belt, you need to measure the:

  • "Track Pitch" - the distance from one lug (the part that engages with the drive wheel) to the next
  • "Track Segment Count" - how many lugs there are on the track
  • "Belt Thickness" - how thick the track would be if you removed the lugs and the tread
